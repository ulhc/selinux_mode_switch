# SELinux 模式一键切换 (SELinuxSwitch)
一个适用于 KernelSU 的轻量级模块，通过音量按键快速切换 SELinux 运行模式。

## ✨ 功能特性
- 📊 **实时显示**当前 SELinux 状态
- 🔑 **按键操作**，无需输入命令
  - 音量上键 = 切换至**强制模式 (Enforcing)**
  - 音量下键 = 切换至**宽容模式 (Permissive)**
  - 电源键 = 取消操作
- ⏱️ **智能倒计时**，10秒无操作自动退出
- 🛡️ **专属适配**，仅支持 KernelSU 环境

## 🚀 使用方法
1. 在 KernelSU 中刷入本模块
2. 安装完成后，在模块列表点击运行
3. 根据屏幕提示，按**音量键**进行切换
